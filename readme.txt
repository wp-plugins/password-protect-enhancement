Enhance the password protection for posts and pages. Optional setting to display an excerpt, and change the current 'This post is password protected. To view it please enter your password below:' text. Based upon an original idea at http://simbo.de/

== Description ==

The plugin adds a new section to post and page edit pages 'Password Protect enhancement'. This may not be next to the 'Password Protect' section, so please check carefully.


== Installation ==

Download the plugin, upload to your Wordpress plugins directory and activate.
